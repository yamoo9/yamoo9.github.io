개인 사업자 등록 절차

전자상거래에 필요한 사업자 등록 절차를 정리합니다.

  1. 사업자 등록증 발급
  2. 사업자 통장 개설
  3. 구매안전서비스(에스크로) 이용 확인증 발급
  4. 통신판매업 신고증 발급
  5. 쇼핑몰 / 오픈마켓(예: 스마트스토어) 개설

개인 사업자로 알아야 할 사항

사업을 준비하려면 필요한 것도 많고, 알아야 할 것도 많다. 공부한 내용을 정리해본다.

A. 세금계산서는 사업자만 발행 가능

사업자 등록증 발급이 필요하다. 세금 계산서는 일반인이 아니라,
사업자가 발행하는 것이기 때문이다.

B. 홈택스에서 전자 계산서 발행 시 필요 사항

전자(세금)계산서 보안카드는 사업자 통장이 필요하므로 사업자 통장을 만들어야 한다.

  • 사업자 통장
  • 전자(세금)계산서 보안카드 또는
    전자(세금) 계산서 발급용 공인인증서

C. 사업자 공인인증서 발급

  • 은행 업무용 공인인증서 (4,400원/년)
  • 전자(세금) 계산서 발급용 (4,400원/년) → 전자(세금)계산서 보안카드 대체 가능 (세무서 발급, 무료)
  • 통합! 범용공인인증서 (전자세금계산서 발급 기능이 포함된 범용공인인증서는 비쌈. 110,000원)

D. 사업 통장 쪼개기

사업 자금 관리를 위해 지출, 수입, 세금으로 분류하여 통장을 관리하는 것이 좋다. → 참고

  1. 지출용 통장 (초기 자금 및 출금 전용)
  2. 수입 전용 통장 (판매 대금 입금용) ← 사업자 통장
  3. 세금 전용 통장 (세금만 보관) ← 부가가치세 (VAT) 등 보관용

1. 사업자 등록증 발급

국세청 홈택스 또는 모바일 손택스를 통해 사업자 등록증 신청.
※ 개인 공인 인증서 또는 모바일 지문 인식 등 필요.

2. 사업자 통장 개설

주 거래 은행에 방문하여 통장을 개설한다.

  • ‘기업,대출 창구’ 번호표를 뽑아 대기해야 한다.
  • 현금 준비(10,000원 이상)
  • OTP 토큰 또는 카드 발급 필요. (토큰 5,000원 / 카드 10,000원)

준비사항

  • 사업자 등록증
  • 신분증
  • 도장 (사인으로 대체 가능)
  • 사업 홈페이지 (https://yamoo9.github.io/EUID)

✔︎ 거래 실적이 없는 경우, 거래 내역 제한 통장을 개설해야 한다. (인터넷 뱅킹 30만원, 창구 100만원, 인출 30만원)
✔︎ 사업자 통장 개설 시, 인터넷 뱅킹도 신청한다.
✔︎ 사업자 통장 개설 시, 10,000원 미리 입금 필요 (추후 공인인증서 비용에서 나가야 하므로)

3. 전자(세금)계산서 보안카드 발급

세무서 방문 후, 전자(세금)계산서 보안카드 사용자 신청서를 작성해 발급 받는다. → 참고

4. 사업자용 공인인증서 발급

계산서 발행을 위해서는 사업자용 공인인증서가 필요하다.
계산서는 사업자가 발행하는 것이기 때문이다.

국민은행 웹 사이트 접속

공인인증센터 → 기업용 인증서 발급 CLICK!

은행 업무용 공인인증서
홈택스에서 사업자로 계산서 발행 조건이 주어질 뿐, 발급은 가능하지 않다.

전자(세금)계산서 보안카드
계산서 발행 시에 필요하다. 세무서를 방문해 보안 카드를 만들 수 있다.

✔︎ 공인 인증서를 저장할 매체(예 USB 등) 준비!
✔︎ 공인 인증서 발급 세금 계산서 신청

기업용 인증서 등록

공인인증센터 → 공인인증서 등록 (사업자)

6. 구매안전서비스(에스크로) 이용확인증 발급

국민은행 사이트 접속 → KB에스크로 이체 → 판매자 인증마크 → 판매 인증마크 등록 →
사업자 공인인증서 로그인 → 인증마크 등록 → 구매안전서비스 이용 확인증 발급

7. 통신판매업 신고증 발급

통신판매업 신고증 발급에 필요한 서류

  • 사업자등록증
  • 구매안전서비스(에스크로) 이용확인증
  • 호스트 소재지 주소(사업장 주소지 아님)

정부24 사이트 접속 후 신청. 발급 절차 → 참고

8. 전자 세금계산서 발행

홈택스 → 로그인 → 전자 세금계산서 발급
✔︎ 은행 업무용 공인 인증서는 발급까지는 못한다. 그래서 보안 카드가 필요하다.

전자 세금계산서 일반(사업자)

세금계산서 - 부가가치세가 과세되는 품목을 공급하는 일반 과세자가 사용
계산서 - 면세 품목을 취급하는 면세 사업자가 사용
             (아파트 85m2의 경우 면세 품목, 초과할 경우 건물 분의 10% 부가세 부과)

✔︎ 계산서 발급을 위해 보안 카드를 사용해 번호를 입력해야 한다. 참고
